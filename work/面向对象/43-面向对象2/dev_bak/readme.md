## 基本数据类型和对象类型的存储
+ 基本数类型变量和值都在栈内存
+ 对象类型变量在栈内存,值内容在堆内存. 变量保存的是对该对象在堆内存中的引用.
+ 如果多个变量同时引用同一个变量,修改一个变量的对象的属性,会导致所有的变量的对象的属性都发生变化,因为他们保存的是通过一个对象的引用.

## 基本数据类型和对象类型作为参数传递
+ 基本数据类型作为参数传递,函数内部会创建一个副本来使用. 所以在函数内修改改值,不会影响函数外部的变量的值.
+ 对象类型作为参数传递,函数内部修改该对象的属性,将直接影响到函数外部的该对象.
